################################################################################################################################
#
#                              Create your own console comic strip  !
#
#    Basic idea:    A comic strip consists of one or more sequences. Each sequence consists
#                   of a number of frames. Each frame consists of the following parts:
#
#    +---------------------------------------------------------------------------------+
#    |                                                                                 |
#    |                                   Static Header                                 |
#    |                                                                                 |
#    +---------------------------------------------------------------------------------+
#    |                                                                                 |
#    |                                  Dynamic Area                                   |
#    |                                                                                 |
#    |                    +---------------------------------------+                    |
#    |                    |S                                      |                    |
#    |                    |              Image Area               |                    |
#    |                    |                                       |                    |
#    |                    +---------------------------------------+                    |
#    |                                                                                 |
#    +---------------------------------------------------------------------------------+
#    |                                                                                 |
#    |                                 Static Footer                                   |
#    |                                                                                 |
#    +---------------------------------------------------------------------------------+
#    
#    The whole frame is subdivided into two static areas which are constant for all sequences.
#    The dynamic area is constant for all frames in each sequence. In case there is no image 
#    defined or the image is completely located outside the dynamic area, the dynamic area 
#    contains only the defined background. Otherwise the image represents the foreground  
#    that hides the background. 
#
#    The foreground may be embedded into the background in one of three ways. First, there is 
#    no transparancy, i.e. spaces in the foreground hides the background. Second, each white space
#    in the foreground is replaced by the corresponding background char (Full transparancy). 
#    Third, the white space is only replaced at the margin of the image. This means: If the
#    image is represented by a rectangle surrounded by white space, only the white space
#    outside the rectangle would be replaced but not the whitespace within the rectangle.
#
#    The foreground (the image) can move within the dynamic area (but not into the static areas!). 
#    The movement can be in any direction: horizontally, vertically or both in different speeds.
#
#    The foreground consists of a series of single image pictures that are rotationally used
#    when drawing the frames of a specific sequence. For each sequence you specify the image area's
#    start point (see 'S' in the image area above) defined by
#    - the horizontal start position (position of first char in dynamic area) and
#    - the vertical start position (number of the first image line withing the dynamic area).
#
#    For more information about the MOGLiCC meta model (i.e. the uses of model, classes, attributes, 
#    metainfos and variables) see MOGLiCC_JavaBeanModel.txt.
#
#    For more information about the console comic strip model see comments below.
#
#    Final hint: avoid using functional characters such as | or >.
#
################################################################################################################################
